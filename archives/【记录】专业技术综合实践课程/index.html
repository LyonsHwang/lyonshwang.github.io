【记录】专业技术综合实践课程

Lyons Hwang
2021-07-01
HTML, Java, JavaScript

这是一门讲网页设计的课……

张永民老师在从教以前似乎是做过这方面的工作……课程所用的网站小溪网是他自己搭出来的……虽然用户界面很普通……虽然好像每个功能都很基础……但需要的功能也都有了……在学习中几乎没有遇到过什么问题……除了无法远程连接小溪网的数据库……

我没怎么认真上这门课……而张永民老师十年如一日地认真上课……老师做了非常好的课件……我通过阅读课件系统地学习了网页相关的知识……比直接搜文档体验好一点……不过……因为所有课程作业的答案都可以在网上找到……所以不看课件也可以有正常的进度……又因为闭卷考试前发了答案……所以不看课件也可以有正常的成绩……

闭卷考试的题量好大……早知道像大家一样打印答案或者摸出手机在考场上抄了……最后看大家抄得挺明目张胆的……老师在讲台上就像没看到一样……

问张永民老师怎么看作弊这件事……他说他根本没发现有人作弊……最多只是有人交头接耳……我说挺多的……而且有些人直接打印答案来抄的……他好像从教这么多年第一次听说这种事一样震惊……

我问他为什么这个题量不开卷……他说因为希望学生能复习……开卷就不会复习了……我不知道怎么评价他的这种想法……我也不知道怎么评价他的这种人生……在大学里教书又不做研究……不知道怎么评价这门课……用的东西挺古老的……打工应该不会用上……

我觉得教学不是一件值得钻研的事情……可以积累一些好的例子……可以做一些好的课件……但是没有什么特别值得钻研的东西……对自己所面对的东西有个比较客观的认知比较好……但是……感觉张永民老师也是挺自信的一个人……一个对自己做出的东西很满意的人……一个非常希望学生能走完他走过的路的人……

遇到有着骄傲的人没法讲任何超出他骄傲领域的东西……不想再和老师有任何沟通了……老师不像是这个时代的人……有点害怕自己以后变成老师这样……害怕自己的博客在后人的眼中和小溪网是一个艺术水准的……然而这门课也学到了很多东西……希望最后成绩比较好看……希望最后成绩比较好看……希望最后成绩比较好看……
